## Why

当前企业管理后台使用统一的默认主题风格和平台 logo，企业客户无法自定义界面颜色和显示自身品牌标识，缺乏品牌归属感和个性化体验。为提升 SaaS 平台的客户满意度和品牌认同感，需要为每个企业提供主题换肤和 logo 自定义功能。

## What Changes

- 企业租户可在管理后台上传、替换、删除企业 logo
- 企业 logo 在管理后台顶部导航栏、登录页等位置显示
- 提供预设主题方案（如默认蓝、科技绿、活力橙、深邃紫等）供企业选择
- 支持自定义主题颜色（主色、辅助色）
- 主题和 logo 配置按租户隔离存储
- 管理后台登录页根据企业域名或租户 ID 展示对应企业的 logo 和主题
- 前端动态加载主题 CSS，无需刷新页面即可切换主题效果

## Capabilities

### New Capabilities
- `enterprise-branding`: 企业品牌配置（logo 上传、主题配置）
- `tenant-theme`: 租户主题管理（预设主题、自定义颜色、动态加载）

### Modified Capabilities
- `enterprise-admin`: 增加品牌配置菜单入口，顶部导航栏显示企业 logo
- `user-management`: 企业登录页根据租户显示对应品牌

## Impact

- **后端**: 需新增租户品牌配置表（logo URL、主题配置），新增品牌配置 API
- **前端**: 管理后台需新增品牌配置页面，实现主题动态切换机制，登录页需支持按租户展示品牌
- **存储**: 需要 OSS/文件存储服务存储企业 logo 图片（或对接现有 logo 管理平台）
- **多租户**: 品牌配置通过 tenant_id 隔离，企业间互不影响
