# Carbon Point v2.0 发布说明

## 版本概述

本次发布带来了统一的资源-套餐-权限架构，实现了平台配置、企业联动的完整解决方案。

## 主要新功能

### 1. 统一资源模型

- 平台资源统一管理（功能产品、商城商品、功能点、权限组）
- 资源元数据支持
- 资源状态管理（启用/停用/废弃）

### 2. 灵活套餐管理

- 可视化套餐创建
- 套餐-资源关联配置
- 必选/可选资源支持
- 套餐级默认配置

### 3. 企业套餐分配

- 企业-套餐绑定
- 套餐变更历史
- 租户资源自动同步

### 4. 动态菜单系统

- 基于套餐的菜单显示
- 基于权限的菜单过滤
- 菜单树构建优化

### 5. 企业资源配置

- 企业级资源启用/停用
- 企业自定义资源配置
- 配置继承与覆盖

## 技术架构改进

### 后端

- 新增 ResourceRegistry 资源注册表
- 新增 TenantPackageService 租户套餐服务
- 新增 MenuService 动态菜单服务
- 统一的 API 设计

### 前端

- 平台侧: 统一套餐管理页面
- 平台侧: 企业套餐分配页面
- 企业侧: 动态菜单布局
- 企业侧: 产品功能配置页面

### 数据库

- 11 个新表
- 完整的迁移脚本
- 向后兼容设计
