# 企业端 E2E 测试用例

## 1. 登录模块（LoginPage）

### 1.1 正常登录流程
- **Given** 用户在登录页面
- **When** 输入正确的手机号和密码，点击"登录"
- **Then** 登录成功，页面跳转到 `/dashboard`，显示用户信息

### 1.2 密码错误
- **Given** 用户在登录页面
- **When** 输入正确的手机号和错误的密码，点击"登录"
- **Then** 显示错误提示"用户名或密码错误"，停留在登录页

### 1.3 未注册用户登录
- **Given** 用户在登录页面
- **When** 输入未注册的手机号和任意密码，点击"登录"
- **Then** 显示错误提示，停留在登录页

### 1.4 手机号格式校验
- **Given** 用户在登录页面
- **When** 输入格式错误的手机号（如 12345678901、abc12345678）
- **Then** 显示"请输入正确的手机号"校验提示

### 1.5 必填字段校验
- **Given** 用户在登录页面
- **When** 不输入任何内容直接点击"登录"
- **Then** 显示"请输入手机号"和"请输入密码"提示

### 1.6 记住用户名
- **Given** 用户在登录页面
- **When** 勾选"记住用户名"，输入正确凭据登录
- **Then** 登录成功后，下次访问登录页时自动填充用户名

## 2. 仪表盘（Dashboard）

### 2.1 仪表盘数据加载
- **Given** 企业管理员已登录
- **When** 访问仪表盘页面
- **Then** 正确显示 4 个统计卡片（今日签到、今日积分、活跃成员、本月兑换）和趋势图表

### 2.2 打卡趋势图
- **Given** 企业管理员已登录
- **When** 访问仪表盘页面
- **Then** 显示打卡趋势面积图，默认展示最近 30 天数据

### 2.3 积分趋势图
- **Given** 企业管理员已登录
- **When** 访问仪表盘页面
- **Then** 显示积分趋势柱状图，展示发放积分与消耗积分对比

### 2.4 热门商品排行
- **Given** 企业管理员已登录
- **When** 访问仪表盘页面
- **Then** 显示热门商品排行表格，前三名显示金/银/铜色排名标识

### 2.5 自然语言查询
- **Given** 企业管理员已登录且仪表盘数据已加载
- **When** 在 NaturalLanguageQuery 组件输入"今日签到数据"并查询

### 2.6 仪表盘无数据状态
- **Given** 企业刚开通，无任何打卡和积分数据
- **When** 访问仪表盘页面
- **Then** 显示空状态或"暂无数据"占位，不报错误

## 3. 成员管理（Member）

### 3.1 成员列表加载
- **Given** 企业管理员已登录
- **When** 访问成员管理页面
- **Then** 显示成员列表，包含头像、姓名、手机号、积分、等级、状态

- **Given** 企业管理员已登录
- **When** 点击"添加成员"，填写手机号和姓名，点击确定
- **Then** 成员新增成功，列表刷新，新成员出现在列表中

### 3.3 添加成员 - 必填校验
- **Given** 企业管理员已登录，打开添加成员弹窗
- **When** 不填写任何字段直接点击确定
- **Then** 显示"请输入手机号"和"请输入姓名"校验提示

### 3.4 添加成员 - 手机号格式错误
- **Given** 企业管理员已登录，打开添加成员弹窗
- **When** 输入格式错误的手机号并提交
- **Then** 显示"请输入正确的手机号"提示

### 3.5 搜索成员
- **Given** 成员列表已有数据
- **When** 在搜索框输入成员姓名/手机号
- **Then** 列表实时过滤，显示匹配的成员

### 3.6 批量导入成员
- **Given** 企业管理员已登录
- **When** 点击"批量导入"，上传正确格式的 .xlsx 文件
- **Then** 显示导入进度，导入成功后列表刷新

### 3.7 批量导入 - 文件格式错误
- **Given** 企业管理员已登录
- **When** 点击"批量导入"，上传非 Excel 文件（如 .txt）
- **Then** 显示文件格式错误提示，上传被拒绝

### 3.8 禁用成员
- **Given** 成员列表中有正常状态的成员
- **When** 点击该成员的"停用"按钮
- **Then** 成员状态变更为"停用"，显示红色标签，禁用后该成员无法登录打卡

### 3.9 启用已禁用成员
- **Given** 成员列表中有停用状态的成员
- **When** 点击该成员的"启用"按钮
- **Then** 成员状态变更为"正常"，显示绿色标签

### 3.10 邀请链接生成
- **Given** 成员列表中有正常状态的成员
- **When** 点击该成员的"邀请"按钮
- **Then** 生成邀请链接并复制到剪贴板，显示"已复制到剪贴板"提示

## 4. 角色权限（Roles）

### 4.1 角色列表加载
- **Given** 企业管理员已登录
- **When** 访问角色管理页面

### 4.2 创建自定义角色
- **Given** 企业管理员已登录
- **When** 点击"添加角色"，填写角色名称和描述，选择权限，点击确定
- **Then** 角色创建成功，列表中出现新角色

### 4.3 创建角色 - 必填校验
- **Given** 企业管理员已登录，打开添加角色弹窗
- **When** 不填写角色名称直接点击确定
- **Then** 显示"请输入角色名称"校验提示

### 4.4 超管角色不可编辑
- **Given** 企业管理员已登录
- **When** 查看角色列表，点击超管角色的"编辑"按钮
- **Then** 编辑按钮不存在或点击后无反应，超管角色不可修改

- **When** 点击某个角色的"查看权限"按钮

- **When** 点击某个自定义角色的"删除"按钮
- **Then** 弹出确认框，确认后角色删除成功，从列表中移除

### 4.8 删除角色 - 超级管理员不可删除

- **When** 尝试删除超管角色（点击删除或通过弹窗）
- **Then** 删除操作被禁止，超管角色保留在列表中

### 4.9 未授权权限灰显
- **Given** 企业管理员已登录，打开编辑权限弹窗
- **When** 查看权限树
- **Then** 当前套餐未授权的权限项显示为灰色禁用状态，悬停显示"平台未授权"提示

## 5. 积分管理（Points）

### 5.1 按手机号查询用户积分
- **Given** 企业管理员已登录
- **When** 输入已注册成员的手机号，点击查询
- **Then** 显示用户信息卡片（头像、姓名、手机号）、总积分、可用积分、用户等级，以及积分流水列表

### 5.2 查询不存在的用户
- **Given** 企业管理员已登录
- **When** 输入未注册的手机号，点击查询
- **Then** 显示"未找到该用户"提示，用户信息区为空

### 5.3 发放积分
- **Given** 企业管理员已登录，已查询到某用户积分信息
- **And** 当前管理员具有 `points:adjust` 权限
- **When** 点击"发放积分"，填写积分数量（如 100）和说明（如"活动奖励"），点击确定
- **Then** 积分发放成功，显示"积分发放成功"提示，总积分和可用积分增加，流水列表刷新

### 5.4 发放积分 - 无权限
- **Given** 企业管理员已登录，已查询到某用户积分信息
- **And** 当前管理员不具有 `points:adjust` 权限
- **When** 查看积分管理页面
- **Then** "发放积分"和"扣减积分"按钮不显示

### 5.5 发放积分 - 必填校验
- **Given** 企业管理员已登录，打开发放积分弹窗
- **When** 不填写积分数量或说明直接点击确定
- **Then** 显示相应的必填校验提示

### 5.6 发放积分 - 积分值校验
- **Given** 企业管理员已登录，打开发放积分弹窗
- **When** 输入积分数量为 0 或负数
- **Then** 显示"积分必须大于 0"提示或 InputNumber 自动阻止输入

### 5.7 扣减积分
- **Given** 企业管理员已登录，已查询到某用户积分信息
- **And** 用户可用积分充足
- **When** 点击"扣减积分"，填写积分数量和说明，点击确定
- **Then** 积分扣减成功，显示"积分扣减成功"提示，总积分和可用积分减少，流水列表刷新

### 5.8 扣减积分 - 余额不足
- **Given** 企业管理员已登录，已查询到某用户积分信息
- **And** 用户可用积分为 0
- **When** 输入扣减积分值大于 0，点击确定
- **Then** 显示错误提示，积分扣减失败

### 5.9 积分流水分页
- **Given** 企业管理员已登录，已查询到某用户积分信息
- **And** 该用户有超过 10 条积分流水
- **When** 翻到积分流水列表的第 2 页
- **Then** 显示第 2 页的流水记录

## 10. 企业配置（Branding）

### 10.1 品牌配置加载
- **Given** 企业管理员已登录
- **When** 访问企业配置页面
- **Then** 显示当前的 Logo、主题配置，以及右侧实时预览面板

### 10.2 上传企业 Logo
- **Given** 企业管理员已登录
- **When** 上传一张符合要求的图片（PNG/JPG，< 500KB）作为 Logo
- **Then** Logo 上传成功，预览面板中 Logo 更新显示

### 10.3 Logo 文件大小超限
- **Given** 企业管理员已登录
- **When** 上传一个超过 500KB 的图片文件作为 Logo
- **Then** 显示文件大小超限提示，上传被拒绝

### 10.4 Logo 文件类型错误
- **Given** 企业管理员已登录
- **When** 上传一个非图片文件（如 .pdf）作为 Logo
- **Then** 显示文件格式错误提示，上传被拒绝

### 10.5 选择预设主题
- **Given** 企业管理员已登录
- **When** 选择一个预设主题（如"森林绿"）
- **Then** 预览面板中配色方案切换为对应主题的蓝绿色调

### 10.6 自定义主题颜色
- **Given** 企业管理员已登录
- **When** 切换到自定义模式，使用颜色选择器设置主色和辅色
- **Then** 预览面板实时反映自定义配色

### 10.7 保存品牌配置
- **Given** 企业管理员已登录，已修改 Logo 或主题
- **When** 点击"保存"按钮
- **Then** 配置保存成功，显示"保存成功"提示

### 10.8 删除 Logo
- **Given** 企业管理员已登录，已有上传的 Logo
- **When** 点击"删除 Logo"按钮
- **Then** Logo 被删除，预览面板中 Logo 恢复为默认占位符

### 10.9 实时预览更新
- **Given** 企业管理员已登录，打开企业配置页面
- **When** 修改主题颜色（不保存）
- **Then** 预览面板实时更新，无需保存即可看到效果
